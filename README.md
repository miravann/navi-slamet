# SLAMET-NAVI

## Nama Produk
SLAMET-NAVI

## Deskripsi Produk
SLAMET-NAVI adalah aplikasi WebGIS Mobile yang dirancang untuk mendukung aktivitas pendakian Gunung Slamet. Aplikasi ini menyediakan navigasi peta, informasi cuaca real-time, agenda pendakian, serta fitur komunikasi (chat) antar pendaki untuk meningkatkan keamanan dan pengalaman selama pendakian.

## Komponen Pembangun Produk
- React Native (Expo) + TypeScript
- Expo Router (file-based routing)
- Firebase:
  - Authentication
  - Realtime Database
- React Native Maps
- React Native Calendars
- React Native Chart Kit
- OpenWeatherMap API untuk data cuaca real-time

## Sumber Data
- Dataset jalur & waypoint (file GPX / kontribusi basecamp / komunitas)
- Data cuaca dari OpenWeatherMap API
- Data pengguna, agenda, dan aktivitas dari Firebase Realtime Database & Firestore
